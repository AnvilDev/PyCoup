# PyCoup
Online version of popular card games, Coup and Coup: Rebellion, along with all of their expansions, written in beautiful Python 3.

The goal is to build an API for a web-client and possibly applications, both mobile and desktop.

# Contributing
The cards have to be completed, they are located in cards/ and organized based on their respective games and role.
These cards pull their actions from a set of basic actions defined in actions.py.
